Proyecto Hibernate 🗃️

Aplicación Java con base de datos relacional usando Hibernate ORM

¿De qué trata este proyecto?

Este proyecto fue desarrollado en Java con el objetivo de aprender y aplicar los conceptos fundamentales de Hibernate ORM, una herramienta de mapeo objeto-relacional (ORM) que permite conectar objetos Java con tablas de bases de datos relacionales.

🔧 Funcionalidades Clave

  • Conexión con base de datos MySQL usando Hibernate
  • Gestión de entidades como Usuario, Producto y Pedido
  • Relaciones OneToMany y ManyToOne correctamente mapeadas
  • Operaciones CRUD completas: Crear, Leer, Actualizar, Eliminar
  • Control de errores con manejo de transacciones
  • Archivo de configuración hibernate.cfg.xml

Visualización de interacciones por consola con la base de datos.

💡 Curiosidades sobre Hibernate

  • Hibernate traduce automáticamente objetos Java en registros de tabla SQL.
  • Utiliza una caché de primer y segundo nivel para mejorar el rendimiento.
  • Permite usar HQL (Hibernate Query Language), un lenguaje similar a SQL pero orientado a objetos.
  • Muy usado en aplicaciones empresariales junto con Spring.
