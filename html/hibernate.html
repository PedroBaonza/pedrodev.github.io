Proyecto Hibernate 🗃️

CRUD completo en Java con Hibernate + relaciones y configuración JPA

🔍 Descripción General

Este proyecto consiste en una aplicación Java que utiliza Hibernate como ORM para realizar operaciones CRUD completas sobre una base de datos relacional. Es ideal para entender cómo funcionan las entidades, las relaciones entre tablas y la configuración mediante JPA.

⚙️ Características Técnicas

  • Implementación de clases entidad con anotaciones JPA.
  • Conexión con base de datos MySQL mediante archivo hibernate.cfg.xml.
  • Relaciones OneToMany y ManyToOne entre entidades.
  • Consultas HQL y manejo de sesiones Hibernate.
  • Interfaz de consola simple para probar operaciones CRUD.
